# Instrument Atlas v0.1.0

Initial alpha release of Instrument Atlas, an open metadata layer for musical instruments.

## Included

- Stable instrument IDs
- Instrument metadata for search and application use
- Taxonomy data
- Instrument packs
- Locale-ready instrument names and aliases
- JSON artifacts for consuming applications
- Validation, build, and packaging scripts

## Purpose

Instrument Atlas provides reusable instrument metadata for music apps, audio tools, AI music systems, game audio tools, DAWs, and creative coding projects.

Instrument Atlas Sounds can be used alongside this repository to map instrument IDs to playable, license-aware sound sources.

## Notes

This is an early alpha release.

The schema and metadata are expected to evolve, but the project aims to keep instrument IDs stable whenever possible.
